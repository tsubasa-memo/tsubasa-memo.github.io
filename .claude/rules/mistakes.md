# mistakes.md — 過去のミスと対策

## Case 1: ペルソナに設定にない属性を追加した

状況: 記事内でツバサの勤務先を「メーカー勤務」と書いてしまった。
設定にない属性を推測で補完した。

→ 対策: ペルソナの設定はwriting-style.mdに明記した4項目のみ。
不明な点が生じたら確認を取ること。絶対に推測で埋めない。

## Case 2: 東京レタッチ・写真加工屋さんを個人向けとして紹介した

状況: レタッチ記事で「1枚から依頼できる」文脈で名前を出してしまった。
どちらも法人・企業向けサービスなのに個人向けとして紹介した。

→ 対策: 東京レタッチ・写真加工屋さんは個人向けの文脈では絶対に名前を出さない。
個人向けとして紹介できるのはリメージ・レタッチインク・ひとフォト・BELLEFOTO等。
