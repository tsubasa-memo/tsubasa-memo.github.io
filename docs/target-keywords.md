## キーワードの配置ルール

### titleタグ（32〜40文字）
```
写真レタッチの料金相場まとめ｜20社以上の価格を1ページに集約【2026年版】
```
含まれるキーワード：レタッチ、料金、相場

### meta description（約120文字）
```
写真レタッチ・画像加工を外注する場合の費用相場を20社以上のサービスから調査。肌レタッチ、色補正、切り抜き、合成など作業内容別の1枚あたりの料金目安と、個人でも依頼できるサービスを比較しました。
```
含まれるキーワード：レタッチ、画像加工、外注、費用、相場、肌レタッチ、色補正、切り抜き、合成、料金、個人

### h2見出しにキーワードを自然配置
- h2: レタッチ料金の相場感（作業内容別）← 「レタッチ 料金 相場」
- h2: サービス別の料金比較表 ← 「料金 比較」
- h2: 料金を左右する要素 ← 「料金」
- h2: レタッチ費用を安く抑えるコツ ← 「レタッチ 費用 安い」
- h2: よくある質問 ← ロングテールをFAQで拾う

### FAQ構造化データに含めるキーワード
各質問文にキーワードを自然に含める：
- 「写真レタッチ1枚あたりの料金相場は？」← レタッチ 1枚 いくら
- 「個人でもレタッチを外注できますか？」← 写真修正 依頼 個人
- 「レタッチの費用を安く抑える方法は？」← レタッチ 費用 安い
- 「肌レタッチの相場はいくらですか？」← 肌レタッチ 料金

## 注意：キーワードの詰め込みはしない
- 同じキーワードをh2に2回以上使わない
- 本文では自然な文脈で使い、不自然な繰り返しは避ける
- 「レタッチ」「写真修正」「画像加工」「フォトレタッチ」は同義語として散らして使う（同じ言葉を連打しない）
